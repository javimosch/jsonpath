# jsonpath
CLI utility. Written in Go.
```bash
curl -LO https://github.com/javimosch/jsonpath/releases/latest/download/jsonpath-linux-amd64
chmod +x jsonpath-linux-amd64; sudo mv jsonpath-linux-amd64 /usr/local/bin/jsonpath
```

---

This tool is a plugin for [supercli](https://github.com/javimosch/supercli) — an AI-friendly, config-driven dynamic CLI platform.
